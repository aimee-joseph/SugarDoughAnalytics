**Sugar Dough Sales Insights Dashboard**

This project is an interactive sales analytics dashboard created for a fictional bakery brand, Sugar Dough, to analyze business performance and customer trends. The dashboard provides insights into total sales, profit, order volume, customer metrics, and cake sales distribution using visually engaging charts and KPI cards.

Features: 
- Sales performance overview with key KPIs
- Day-wise order trend analysis
- Cake category sales distribution
- Quarterly growth and QoQ analysis
- Customer and revenue insights
- Clean bakery-themed dashboard design

Tools & Technologies:
- Power BI
- Data Visualization
- DAX Measures
- Data Cleaning & Transformation

Key Insights:
- Identified top-performing cake categories
- Analyzed weekly sales fluctuations
- Measured profitability and revenue per order
- Evaluated quarterly sales growth trends

This dashboard was designed to demonstrate business intelligence and storytelling through data visualization in a retail/bakery sales environment.
